# VRT Data Graphics Generator

A web-based tool for creating data infographics and visualizations, designed with the VRT NWS brand identity.

## Features

- **Multiple Chart Types**: Bar charts (vertical/horizontal), line charts, pie charts, and donut charts
- **Color Palette**: Customizable primary, secondary, and highlight colors
- **Background Upload**: Upload images or videos as chart backgrounds
- **Motion Controls**: Set animation timing for chart entrance and exit
- **Custom Data Input**: Enter X and Y axis data directly
- **Easing Curve Editor**: Visual bezier curve editor for animation timing
- **Export Options**: Export as PNG, JPG, or video formats (MP4, MOV)
- **Timeline Playback**: Preview animations with full timeline controls

## Usage

1. Open `index.html` in a web browser
2. Select your chart type from the dropdown
3. Customize colors using the color pickers
4. Enter your data in the X and Y axis input fields
5. Add a title, subtitle, and source
6. Upload a background image if desired
7. Adjust animation timing with the motion controls
8. Preview your animation using the timeline
9. Export your graphic in your preferred format
